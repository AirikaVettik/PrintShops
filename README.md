# PrintShops
Estonian Print Shops


## Project setup
```
Be in server folder - npm install
Be in client folder - npm install
```

### Start
```
Be in server folder - npm run dev (backend)
Be in client folder - npm run serve (frontend)
```

### Working on
```
PORT: 3000 (backend)
PORT: 8080 (frontend)
```

### Database
```
Mongo db
```
